# Servo Motor Sweep

Basic servo motor movement through several angles.

## Overview

This Arduino practice was developed in Tinkercad as part of a robotics learning sequence. It focuses on a specific concept and keeps the circuit simple so the behavior can be easily tested and understood.

## Tinkercad Project

[Open this project in Tinkercad](https://www.tinkercad.com/things/82jKlbjW9d6-practica-10-robotica?sharecode=NywuJGA0jfOSZUnl0FENYMCYPECNmh1UeMSE8G20MYM)

## How It Works

The servo motor moves through 0°, 45°, 90°, 135° and 180°, waiting two seconds between each position.

## Components

- Arduino Uno
- Servo motor
- Jumper wires

## Pin Configuration

| Pin | Function |
|---|---|
| `3` | Servo signal pin |

## Learning Objectives

- Use the Servo library
- Control angular position
- Create repeated movement sequences
