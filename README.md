# Arduino Robotics Practices

A collection of beginner Arduino robotics practices developed in Tinkercad. These exercises cover the basic concepts of Arduino programming, electronic components, sensors, actuators and simple automation systems.

## Project description

This repository contains a set of Arduino practices created as part of a robotics learning process. Each practice focuses on a specific concept, starting from basic LED control and gradually introducing buttons, the serial monitor, potentiometers, ultrasonic sensors, buzzers, light sensors, temperature sensors and servo motors.

The projects were designed and simulated using Tinkercad Circuits, making them easy to test, understand and modify without requiring physical hardware.

## Practices included

| Practice | Name | Main concept |
|---|---|---|
| Practice 00 | LED Blink | Basic digital output using the built-in LED |
| Practice 01 | Dual LED Alternation | Alternating two LEDs with digital outputs |
| Practice 02 | Traffic Light Sequence | Simulating a simple traffic light sequence |
| Practice 03 | Button Controlled LED | Reading a digital input to control an LED |
| Practice 04 | Serial Monitor Message | Sending repeated messages through the serial monitor |
| Practice 05 | People Counter Button | Counting button activations and displaying the result |
| Practice 06 | Potentiometer LED Speed | Using analog input to control LED blinking speed |
| Practice 07 | Ultrasonic Buzzer Alert | Measuring distance and activating a buzzer alert |
| Practice 07B | Ultrasonic LED Distance Alert | Showing distance levels using LEDs and a buzzer |
| Practice 08 | Light Sensor Automatic LED | Turning an LED on or off depending on light level |
| Practice 09 | Temperature LED Indicator | Indicating temperature ranges with different LEDs |
| Practice 10 | Servo Motor Positions | Moving a servo motor through different angles |

## How to use

1. Open the desired practice folder.
2. Copy the `.ino` code into the Arduino IDE or Tinkercad Circuits.
3. Build or open the corresponding circuit.
4. Run the simulation and test the behaviour.

## Tinkercad projects

Original simulations were created in Tinkercad. Each practice include its own Tinkercad link inside its folder README or documentation file.

## Learning goals

The goal of this repository is to document the progression of robotics and Arduino learning through small practical exercises. These projects help understand how inputs, outputs, sensors and actuators can be combined to create interactive electronic systems.
