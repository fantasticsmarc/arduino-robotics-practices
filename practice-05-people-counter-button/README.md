# People Counter with Button

Simple people counter using a button and the Serial Monitor.

## Overview

This Arduino practice was developed in Tinkercad as part of a robotics learning sequence. It focuses on a specific concept and keeps the circuit simple so the behavior can be easily tested and understood.

## Tinkercad Project

[Open this project in Tinkercad](https://www.tinkercad.com/things/7QOQVpYuLqx-practica-5-robotica?sharecode=UFtktjc8zL8pXc4i4sBo83QJKSF1khMCtZ8QSBk_WNI)

## How It Works

Each time the button is pressed, the counter increases and the updated number is printed in the Serial Monitor.

## Components

- Arduino Uno
- Push button
- Resistor
- Jumper wires
- Serial Monitor

## Pin Configuration

| Pin | Function |
|---|---|
| `7` | Button input |

## Learning Objectives

- Use variables to store values
- Increment a counter
- Print sensor/input information to the Serial Monitor
