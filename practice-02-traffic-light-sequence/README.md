# Traffic Light Sequence

Traffic light simulation with red, yellow and green LEDs.

## Overview

This Arduino practice was developed in Tinkercad as part of a robotics learning sequence. It focuses on a specific concept and keeps the circuit simple so the behavior can be easily tested and understood.

## Tinkercad Project

[Open this project in Tinkercad](https://www.tinkercad.com/things/5QFnvMkyvQo-robotica-practica-2?sharecode=qzHOQk71x9Rb6jPl5HAoVKY-zpgRTQZvqy7xKwOigGQ)

## How It Works

The red LED stays on, then the yellow LED blinks, and finally the green LED turns on before the cycle repeats.

## Components

- Arduino Uno
- 3 LEDs: red, yellow, green
- 3 resistors
- Jumper wires

## Pin Configuration

| Pin | Function |
|---|---|
| `7` | Red LED |
| `6` | Yellow LED |
| `5` | Green LED |

## Learning Objectives

- Build a basic traffic light sequence
- Use several outputs in a timed cycle
- Represent real-world systems with simple logic
