# LED Blink

Basic LED blinking sequence using Arduino digital output.

## Overview

This Arduino practice was developed in Tinkercad as part of a robotics learning sequence. It focuses on a specific concept and keeps the circuit simple so the behavior can be easily tested and understood.

## Tinkercad Project

[Open this project in Tinkercad](https://www.tinkercad.com/things/jHw51OJLeBK-robotica-practica-0?sharecode=FUHhK--DLcufidV_aSGmyZoYQqFAwRW7fwVzGbRsslQ)

## How It Works

The LED turns on for one second and then turns off for one second, repeating continuously.

## Components

- Arduino Uno
- LED or built-in LED
- Resistor, if using an external LED
- Jumper wires

## Pin Configuration

| Pin | Function |
|---|---|
| `13` | LED output |

## Learning Objectives

- Configure a digital pin as output
- Control an LED using HIGH and LOW states
- Use delay() to create timing intervals
