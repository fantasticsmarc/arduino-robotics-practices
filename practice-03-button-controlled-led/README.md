# Button Controlled LED

LED controlled by a push button input.

## Overview

This Arduino practice was developed in Tinkercad as part of a robotics learning sequence. It focuses on a specific concept and keeps the circuit simple so the behavior can be easily tested and understood.

## Tinkercad Project

[Open this project in Tinkercad](https://www.tinkercad.com/things/0fgxYCr85RL-practica-3?sharecode=R42HRLrtbbO1U4GOQdNb_ywdXWQjhFCZMmMYPCpWvhA)

## How It Works

When the button connected to pin 7 is pressed, the LED on pin 13 turns on. When it is released, the LED turns off.

## Components

- Arduino Uno
- Push button
- LED or built-in LED
- Resistor
- Jumper wires

## Pin Configuration

| Pin | Function |
|---|---|
| `7` | Button input |
| `13` | LED output |

## Learning Objectives

- Read a digital input
- React to button presses
- Use conditional logic with if/else
