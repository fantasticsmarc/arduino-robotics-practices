# Contributing

Thank you for your interest in this repository.

This project is mainly a personal collection of Arduino robotics practices, but suggestions, improvements and ideas are welcome.

## How to contribute

You can contribute by:

- Improving the code comments.
- Suggesting cleaner Arduino code.
- Adding circuit diagrams or screenshots.
- Improving the documentation.
- Reporting mistakes in the explanations.
- Proposing new beginner-friendly robotics practices.

## Code style

Please keep the code simple and easy to understand. These practices are intended for learning, so clarity is more important than advanced optimization.

Recommended style:

- Use descriptive variable names.
- Add short comments when useful.
- Keep each practice focused on one main concept.
- Avoid unnecessary complexity.

## Practice structure

Each practice should follow this structure when possible:

```text
practice-name/
├── practice_name.ino
├── README.md
└── images/
    └── circuit-preview.png
```

## Pull requests

Before opening a pull request:

1. Make sure the code works in Arduino IDE or Tinkercad Circuits.
2. Check that the documentation is clear.
3. Explain what you changed and why.

## Issues

If you find a problem, please open an issue and describe:

- Which practice has the problem.
- What the expected behaviour is.
- What actually happens.
- Any screenshots or error messages, if available.

## Notes

This repository is focused on educational Arduino projects. Contributions should keep the beginner-friendly purpose of the project.
