# Dual LED Alternation

Two LEDs blinking alternately using Arduino digital outputs.

## Overview

This Arduino practice was developed in Tinkercad as part of a robotics learning sequence. It focuses on a specific concept and keeps the circuit simple so the behavior can be easily tested and understood.

## Tinkercad Project

[Open this project in Tinkercad](https://www.tinkercad.com/things/2Lg5zUpfKoL-robotica-practica-1?sharecode=UTVZy3GH0cfi_Iwd3md3n9E_IG8gv00xvGYOT5CtTP0)

## How It Works

One LED turns on while the other remains off. After one second, their states are swapped.

## Components

- Arduino Uno
- 2 LEDs
- 2 resistors
- Jumper wires

## Pin Configuration

| Pin | Function |
|---|---|
| `6` | LED 1 |
| `7` | LED 2 |

## Learning Objectives

- Control multiple digital outputs
- Create alternating light patterns
- Understand basic timing with delay()
